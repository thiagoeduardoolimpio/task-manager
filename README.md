# Task Manager (Python)

**PORTUGUÊS**

## Sobre o projeto

Esse é um projeto simples que fiz para treinar lógica de programação em Python.
A ideia é funcionar como um gerenciador de tarefas no terminal, onde o usuário pode digitar várias tarefas e depois visualizar todas organizadas em uma lista.

-------------------------------

## O que ele faz
	•	Permite digitar várias tarefas
	•	Armazena essas tarefas em uma lista
	•	Mostra todas as tarefas numeradas no final

-------------------------------

## Como usar
	1.	Execute o arquivo:
    
        BASH
        python main.py

    2.	Digite suas tarefas normalmente
	3.	Quando quiser parar, digite sair

-------------------------------

## Exemplo
	TERMINAL:
    • Digite uma tarefa (ou 'sair'): estudar python |
    • Digite uma tarefa (ou 'sair'): ir à academia | 
    • Digite uma tarefa (ou 'sair'): sair |

1. estudar python
2. ir à academia
